Adaptations is a MPA web application thats has CI integrated using Docker containers and Jenkins

The application's tech stack includes:
- ASP.NET Core 2.2
- EntityFrameworkCore
- Automapper
- ASP.NET Identity System
- Bootstrap
- jQuery
- Docker
- Jenkins
- JS
- HTML
- CSS

The application includes:
- Pagination
- Partial views
- Repository pattern
- Testing
